8、糖代谢
  • #概述
    • 糖的生理 #功能
      • 一切生物体所需能量的主要来源
        • 正常情况下机体所需总能量的60~70%来自糖的氧化分 解
      • 储存能量，维持血糖
        • 肝糖原和肌糖原
        • 肝糖原用于维持血糖浓度
      • 为其他物质合成提供原料
        • 可为合成脂肪、蛋白质等提供原料
      • 生物体的结构组分
        • 糖复合物是细胞的结构分子 
      • 参与执行多种生理功能
        • 如：细胞识别、血型物质、信号传递等
    • 糖代谢的概述 
      • 分解代谢（大分子--→小分子）
        • 糖酵解途径
        • 三羧酸循环
        • 磷酸戊糖途径
        • 糖原的分解
      • 合成代谢（小分子--→大分子）
        • 糖异生作用
        • 糖原的合成
        • 光合作用
    • 糖类的消化
      • 淀粉的降解（a-1,4糖苷键 ，a-1,6糖苷键）
        • 水解：淀粉酶、脱支酶和麦芽糖酶
        • 磷酸解：磷酸化酶
          • 耗能少
          • 产物不易扩散到胞外
      • 蔗糖的降解（α-1,2-β糖苷键）
        • 水解：蔗糖酶（转化酶）
          • 葡萄糖
          • 果糖
        • 形成糖核苷酸（UDPG和ADPG）：蔗糖合酶
          • UDPG和ADPG是葡萄糖的活化形式，作为葡萄糖基的供体

    • 糖类的吸收
      • 主动转运（肠粘膜细胞和肾小管）
      • 被动转运（红细胞、肌肉和脂肪组织细胞）

  • 糖的分解代谢
    • 糖酵解途径
      • #概述
        • 糖酵解是生物体最重要的分解代谢途径之一，指细胞在细胞质中，没有氧气的参与下葡萄糖氧化生成丙酮酸，并伴随ATP生成的一系列反应，又称EMP途径
      • 糖酵解的 #过程
        • 第一阶段——由葡萄糖形成F-1,6-2P（-2ATP） 
          • 第一步：（己糖激酶）葡萄糖的磷酸化
          • 第二步：（异构酶）6-磷酸果糖的生成
          • 第三步：（磷酸果糖激酶）1,6-二磷酸果糖的生成
        • 第二阶段——3-磷酸甘油醛的形成
          • 第一步4：（异构酶）1,6-二磷酸果糖的裂解
          • 第二步5：（醛缩酶）磷酸丙糖的同分异构化
        • 第三阶段——3-磷酸甘油醛氧化成3-磷酸甘油酸（+2ATP+2NADH）
          • 第一步6：3-磷酸甘油醛氧化反应（脱氢酶）
          • 第二步7：1,3-二磷酸甘油酸的高能磷酸基团转移反应（激酶）
        • 第四阶段——丙酮酸的形成（+2ATP）
          • 第一步8：3-磷酸甘油酸的变位反应（变位酶）
          • 第二步9：2-磷酸甘油酸的脱水反应（烯醇化酶）
          • 第三步10：磷酸烯醇式丙酮酸的磷酸基团转移
      • 糖酵解产物的去路
        • 无氧条件下
          • 乳酸发酵 :NADH + H+再生为NAD+ 
          • 酒精发酵 :NADH + H+再生为NAD+ 
            • 巴斯德效应：氧可以降低糖类的分解代谢和减少糖酵解产物的积累的现象。
        • 有氧条件下
          • 丙酮酸进入三羧酸循环，彻底氧化成CO2
          • 原核细胞中NADH+H+通过呼吸链产生2.5ATP
          • 真核细胞中，NADH+H+经磷酸甘油穿梭系统进入FADH2呼吸链，产生1.5个ATP
      • 生物学意义
        • 单糖分解代谢一条最重要的基本途径
        • 是机体缺氧情况下获取能量的主要途径
        • 是某些组织或细胞的主要获能方式
        • 是葡萄糖完全氧化分解成CO2和H2O的必要准备阶段
        • 其中间产物为生物合成提供原料
      • 糖酵解的总结
        • 糖酵解几乎是生物的公共途径，一分子葡萄糖氧化成两分子丙酮酸，并把能量以ATP和NADH形式贮存
        • 糖酵解过程有10个酶，全部在胞质中。有10个中间产物，都是磷酸化的六碳或三碳化合物
        • 糖酵解的准备阶段，用ATP把葡萄糖转化为1,6-二磷酸果糖，然后C3和C4间的键断裂生成两分子磷酸三糖
        • 在回报阶段，来自葡萄糖的3-磷酸甘油醛在C1上发生氧化，反应能量以一分子NADH和两分子ATP形式贮存
        • 己糖激酶、磷酸果糖激酶和丙酮酸激酶是糖酵解的限速酶（关键酶），受到变构调控，以保证ATP的不断供给
    • 三羧酸循环
      • 糖的有氧分解：葡萄糖在有氧条件下彻底分解生成CO2和H2O并释放大量能量的过程
      • 丙酮酸氧化脱羧
        • 在线粒体基质中进行
        • 丙酮酸通过内膜上载体转运
        • 不可逆反应
        • 丙酮酸脱氢酶系 
          • E1：丙酮酸脱羧酶（6）（TPP）
          • E2：二氢硫辛酸乙酰基转移酶（24）（硫辛酸和CoA）——核心
          • E3：二氢硫辛酸脱氢酶（6）（FAD和NAD+）
      • TCA循环
      • 柠檬酸循环
      • Krebs循环
      • 线粒体基质
      • 三羧酸循环的化学计量
        • 有 2个碳原子通过乙酰 CoA进入循环，以后有2个碳原子（来自草酰乙酸）通过反应离开循环
        • 产生1分子高能磷酸化合物 GTP， 通过它可生成 1分子ATP（底物水平磷酸化）
        • 消耗 2分子H2O，分别用于合成柠檬酸（水解柠檬酰CoA）和延胡索酸的水化
        • 有4对氢原子通过脱氢反应离开TCA循环，其中3对由NADH携带，1对由FADH2携带
      • 丙酮酸完全氧化的化学计量
      • 三羧酸循环的生物学意义
        • 为生物体提供能量，是体内主要产生ATP的途径
        • 是生物中的燃料分子最终氧化的共同途径
        • 是分解代谢和合成代谢途径的枢纽
        • 三羧酸循环产生的CO2，一部分排出体外，其余部分供机体生物合成需要
        • 三羧酸循环是中心代谢途径
    • 磷酸戊糖途径（PPP或HMS）
      • #概述
        • 细胞内重要的分解代谢支路。动物体中有30%的葡萄糖通过HMS途径分解
        • 由6-磷酸葡萄糖开始
        • 全程无ATP生成
        • 发生在细胞质中
      • #过程
        • 不可逆的氧化阶段——糖的脱氢、脱羧
        • 可逆的非氧化阶段——糖的相互转化（分子重排） 
      • 磷酸戊糖途径的 #生物学意义 
        • 产生大量NADPH，为细胞各种合成反应提供还原力
        • 产生的磷酸核糖和中间产物为许多化合物的合成提供原料
        • 与光合作用联系实现某些单糖间的互变
        • 可与EMP、TCA相互补充和配合，增强机体适应性
  • 糖的合成代谢
    • 糖异生作用
      • #概述
        • 指由非糖物质，如甘油、乳酸和各种生糖氨基酸等经过系列反应转化生成葡萄糖或糖原的过程
        • 场所主要为肝，部分在肾中进行。此外大脑、骨骼肌或心肌也进行极少量的糖异生作用
      • 原料
        • 乳酸 → 丙酮酸
        • 甘油 → 磷酸二羟丙酮
        • TCA循环中间产物 → 草酰乙酸 
        • 生糖氨基酸 → 丙酮酸或TCA循环中间产物
      • #过程
        • 不是简单的糖酵解途径的逆转
        • 由丙酮酸激酶催化反应的逆过程(丙酮酸羧化支路 )
        • 由磷酸果糖激酶催化反应的逆过程
        • 由己糖激酶催化反应的逆过程
      • #生物学意义
        • 保证在饥饿情况下血糖浓度的相对恒定
        • 有利于乳酸的利用，更新肌糖原，防止乳酸酸中毒(乳酸循环（Cori循环）)
        • 有利于氨基酸分解和脂肪氧化分解供能
    • 糖原合成
      • #概述
        • 由葡萄糖合成糖原的过程，在细胞质中进行，需要消耗ATP和UTP
        • 肝糖原：分解维持血糖浓度
        • 肌糖原：为肌肉糖酵解提供6-磷酸葡萄糖，通过氧化为肌肉收缩供能.(肌肉中缺乏6-磷酸葡萄糖酶，不能转变为葡萄糖，从而不能补充血糖)
      • UDP
        • 葡萄糖的活化形式
        • 糖原合成时葡萄糖基的供体
      • #过程
        • 糖原合成需要引物
        • 糖原生成素（酪氨酸葡萄糖苷转移酶）合成引物
        • 分支酶作用下引入支链
  • 利用代谢途径生产发酵产品
    • 代谢调节发酵
      • 自然发酵
        • 利用微生物在特定条件下的固有的代谢规律，自然积累某种产品的发酵。如：酒精、乳酸
      • 代谢调节发酵
        • 在代谢途径调节控制理论指导下建立的发酵技术
        • 选育出有关代谢途径旺盛的菌种
        • 阻断代谢途径，积累目标中间产物
        • 补充目标中间产物之后的产物
    • 甘油发酵原理
      • 当有了乙醛作为受氢体，代谢途径的流向就不再朝甘油方向。将受氢体乙醛除去，则势必造成发酵液中甘油的积累.(酒精发酵)
      • 亚硫酸盐法甘油发酵
